This menu allows the user to edit the server configuration (all fields must be filled in):

Log Level indicates the messages to be stored (0=null, 1=error, 2=warning, 3=info)

Log Separator indicates the level to be used as a separator in log files (messages from lower levels will be stored in error.log and the rest in info.log)

Update Frequency indicates how often the server updates the printers

Delete Timeout indicates how often the server deletes the printers with Unreachable status
